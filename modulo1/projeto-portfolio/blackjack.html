Blackjack

Descrição


Ao entrar no site, um prompt se abre perguntando se o usuário deseja iniciar uma partida de Blackjack. Ao serem reveladas suas cartas e uma carta do computador, o usuário tem a opção de comprar mais uma carta apertando "Ok" ou terminar o jogo clicando em "Cancelar". O jogo então termina mostrando se o usuário ganhou ou perdeu e o resultado da soma das cartas do usuário e do computador.

Este projeto foi desenvolvido para um projeto de fixação no curso da Labenu. Para criar este jogo foram utilizadas as tecnologias de JavaScript e HTML.


Tecnologias Utilizadas


HTML | Javascript
