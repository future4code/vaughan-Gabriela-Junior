Hermenalien & Cia

Descrição


Projeto de estilização de um site de uma loja de itens intergalácticos chamada "Hermenealien & Cia". Ao entrar, o usuário tem as opções de ver a página de Controle de Estoque, a página para Adicionar Produto e a página com as informações dos funcionários da loja. Além disso, é possível abrir também a página de Vendas para o funcionário registrar uma nova venda e pode-se acessar também a página de Histórico de Vendas.

Este site foi desenvolvido para um exercício de CSS no curso da Labenu. Para criar este site foram utilizadas as tecnologias de HTML e CSS.
